-Head


Serve para ferramentas css e javascript(fontes, ícones, linkação com css e javascript)

Tag para o algoritimo encontrar a página(<meta></meta>) com o atributo que é a codificação das palavras chaves da tag título(charset)= <meta charset"UTF-8">

Exemplos de tags meta:


<meta name>"description" content="esse site fala sobre....">(legenda do site quando o usuário pesquisa)

<meta name>"author" content="Giulian Fabio Bastos">(nome do autor)

<meta name>"keywords" content="tutorial, html">(palavras chaves)

<meta name>"robots" content="">(pesquisar sobre)

-Tabela


<table>(Coloque o título, lembre-se de usar css o=para estilizar o conteúdo da tabela)
<thead>(Colunas principais em negrito)
<tr>(define linha)
<th>(Aqui coloca-se o que tem em cada coluna)
</th>
</tr>

</thead>

<tbody>(Aqui ficam os dados correspondentes a cada coluna no thead, sem negrito)
<tr>(define linha)
<td>(Aqui coloca-se o que tem em cada coluna, usa-se o atributo colspan ="número de colunas que voce deseja ocupar nessa linha")
</td>
</tr>
</tbody>
</table>

-Css e Javascript


<head>

<link rel="stylesheet" href="caminho do arquivo">(chama o arquivo css dentro do projeto para seu arquivo html, melhor assim as vezes para não tornar um arquivo só muito grande)

<script src="caminho" defer(o defer faz o arquivo ser carregado depois do html, pode-se usar como alternativa o atributo async servindo pra carregar o arquivo off)>(mesma coisa do css acima só que com o arquivo JAVASCRIPT)

</head>

(SCRIPT NO FINAL DO BODY É O CERTO POÍS ARQUIVOS JS PODEM SER PESADOS!)

<script src="caminho">script>(aparece normal)

<script>alert('olá no html') </script>(aperece normal, só que assim o programador que cria)
